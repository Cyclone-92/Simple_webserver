# `Simple_webserver`
## This is a simple webserver that uses node.js/express.js and mongdb.

## Prerequties 
    mongdb
    node js

## Installation
    mkdir -p ~/simple_server
    cd ./simple_server
    git clone https://github.com/Cyclone-92/Simple_webserver.git
    service mongod start
    node server.js
